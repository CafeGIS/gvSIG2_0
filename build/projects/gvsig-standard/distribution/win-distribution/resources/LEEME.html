Contacto: gvsig@gva.es

Requisitos de la aplicación:

Sistema

  • Mínimos: Pentium IV / 512 MB RAM / Targeta grafica compatible con OpenGL 1.5
  • Recomendables: Pentium IV / 1 GB RAM / Targeta Grafica compatible con OpenGL 2.0
  • Sistemas operativos: plataformas Windows y Linux. Probado en XP, Ubuntu Linux y Mac OS X.

Software instalado

(disponible en http://www.gvsig.gva.es o en http://java.sun.com)
  • Máquina Virtual Java 1.5.
  • JAI (Java Advanced Imaging).
  • JAI Image I/O.

Nota: Las librerías JAI y JAI Image I/O deben estar instaladas sobre la Máquina Virtual Java que gvSIG utilice en su ejecución. Si se instala una nueva JVM en el sistema y se usa para ejecutar gvSIG será indispensable instalar las librerías sobre ésta (gvSIG se ejecuta con la JVM que tenga por defecto el sistema operativo; para saber cuál es, ejecutar "java -version" en un interfaz de comandos o consola).
